# Smart Exam System

An Online Examination System built using ASP.NET Core Web API, SQL Server, JWT Authentication, HTML, CSS, JavaScript, and Bootstrap.

## Features

### Student

* Register and Login
* View Available Exams
* Attend Online Exams
* Auto Submit Timer
* View Results
* Email Notification

### Admin

* Create Exams
* Add Questions
* Manage Exams
* Manage Question Bank

## Technologies Used

* ASP.NET Core 8
* Entity Framework Core
* SQL Server
* JWT Authentication
* MailKit
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Project Structure

* Controllers
* Models
* DTOs
* Services
* Data
* wwwroot
